申明：
本应用使用了miniblink框架，官网：http://miniblink.net/


注意：
1.第一次使用请备份游戏目录的nativePC，并且压缩成zip/7z/rar
2.需要安装.NET 4.5框架
3.这是测试版！
4.如果上面有备份，请添加把上面的备份当成MOD添加，并且放在第一位，后面添加的自动覆盖上面
5.检查更新
链接: https://pan.baidu.com/s/1tPDg1i4soaR9NPz4lTpSUA 提取码: 58xg


使用说明：
1.排序下面的会覆盖上面的MOD
2.MOD的冲突，自己看MOD类型判断
3.MOD要打勾才算生效

关于MOD冲突：
只支持单武器和单一套服装
比如你先启用Mod1，然后启用Mod2，Mod2会覆盖了Mod1的文件，然后你禁用Mod2，（覆盖的文件，并不会自动复制回来，你需要点MOD菜单的检查MOD完整性。


MOD手动排序:
关闭MOD盒子，打开Mods/modlist.txt，自己修改顺序并保存为utf8。启动MOD盒子
